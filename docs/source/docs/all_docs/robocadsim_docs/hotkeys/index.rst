Горячие клавиши
=========================================

Горячие клавиши на открытой сцене:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- **Esc:** - открытие меню паузы  
  
- **LeftCtrl + E:** - открытие панели с кнопками
- **LeftCtrl + D:** - открытие панели управления полем
- **LeftCtrl + R:** - перезапуск сцены
